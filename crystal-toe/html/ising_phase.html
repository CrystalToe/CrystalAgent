WHAT YOU'RE SEEING

A 200 × 200 lattice of tiny magnets, each pointing up (blue) or down (red). Every frame, the computer picks random magnets and asks: "should I flip?" If flipping lowers the total energy, it always flips. If flipping raises the energy, it flips with probability e^(−ΔE/kT). This is the Metropolis algorithm — the foundation of all Monte Carlo physics.

THE PHASE TRANSITION

Drag the temperature slider. Three regimes:

T < T_c (cold) — Almost all spins align. One color dominates. This is a ferromagnet. Long-range order emerges spontaneously from local interactions. Start from "↺ Hot" and cool slowly — watch domains grow and merge until one wins.

T = T_c (critical) — Fractal domains at every scale. No single domain wins, but they're not random either. This is scale invariance — the system looks the same zoomed in or out. This is the exact point where order melts into disorder.

T > T_c (hot) — Pure noise. No pattern. Every spin is independent. This is a paramagnet. The thermal energy overwhelms the interaction energy.

WHY N_w = 2 STATES?

Each spin has N_w = 2 possible values: up (+1) or down (−1). This is the weak prime of the Crystal Topos — the same 2 that gives SU(2), the weak force, two truth values in classical logic, and binary digits. The Ising model is the simplest system with a phase transition, and it has exactly N_w states because that's the smallest number that allows symmetry breaking.
