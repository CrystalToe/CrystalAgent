Same Song, Second Verse

the blessed-prime gate governs electron shells

Nuclear magic numbers come from n(n+1) — the pronic sequence. Noble gas electron counts come from n² — the square sequence. Both are the bilinear form n · m: nuclear takes m = n+1, electron takes m = n. The gap is exactly n at every shell. And the same blessed primes gate both.

The Noble Gases — Prime Gate Test

Every noble gas through Radon factors entirely into blessed primes. Oganesson is the first to fail — and relativistic quantum chemistry independently predicts it should not behave as a noble gas.

The Bilinear Parent: n·m

nuclear L1 shell size n · (n + 1) = pronic  →  2, 6, 12, 20, 30, 42, 56, 72, 90, 110
electron shell capacity n · n = square  →  1, 4, 9, 16, 25, 36, 49, 64, 81, 100
gap at each shell n(n+1) − n² = n  — exact, always

Shell-by-Shell Comparison

The nuclear sequence steps one integer ahead at each level. The electron sequence stays put. The difference is always the shell index itself.

Cumulative Sequences — Where They Diverge

The nuclear cumulative (gold) and electron cumulative (blue) track together for n = 1, then separate. The noble gas Z values (blue dots) are the electron cumulative with each capacity doubled for n ≥ 2. The nuclear magic numbers (gold dots) include the spin-orbit correction that kicks in at n = N_c = 3.

Radon and the Heegner Stamp

Z = 86 = 2 · 43. The prime 43 sits directly in the Heegner set H = {1, 2, 3, 7, 11, 19, 43, 67, 163}. It marks the last stable noble gas. The same number-theoretic structure that gives 163 its singular status — the nine imaginary quadratic fields with class number one — stamps the boundary of noble gas stability.

After Radon, the next noble gas position is Z = 118 = 2 · 59. The prime 59 is foreign (59 ∉ H, 4·59−1 = 235 ∉ H). The framework predicts: Oganesson is not a true noble gas. Relativistic DFT agrees — spin-orbit splitting of the 7p shell is expected to make Oganesson reactive, not inert.
