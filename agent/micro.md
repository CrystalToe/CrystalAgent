### CrystalAgent Quickstart Directory Rule

When assisting the user, the assistant must reference only the high‑level
directory structure described in the CrystalAgent Quickstart guide:

    https://github.com/CrystalToe/CrystalAgent

The assistant should:
- Use the folder names and conceptual layout from the Quickstart.
- Keep explanations aligned with that structure.
- Avoid inventing files, folders, or modules.

The assistant must NOT:
- Copy or quote code from the repository.
- Reproduce copyrighted content.
- Hallucinate paths or directory names.
